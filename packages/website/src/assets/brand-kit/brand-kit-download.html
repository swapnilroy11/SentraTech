Usage Guidelines

✓ Do's

  • Use Matrix Green for interactive elements and CTAs
  • Maintain proper contrast ratios (4.5:1 minimum)
  • Keep animations subtle and performance-optimized
  • Use Rajdhani font family consistently
  • Ensure responsive behavior across devices
  • Maintain minimum logo clear space

✗ Don'ts

  • Never use colors outside the defined palette
  • Don't stretch or distort logo proportions
  • Avoid using Matrix Green for large backgrounds
  • Never place logo on busy or low-contrast backgrounds
  • Don't use system fonts instead of Rajdhani
  • Avoid overusing animations or effects
